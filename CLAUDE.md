## What This Project Is

**We Got Us Your IP** (`ip.wegotussome.com`) is a single-page web app that detects and analyses your public IP address. It is entirely client-side with no build process — just HTML, CSS, and vanilla JS, plus a `themes/` directory of JSON files.

**Repo**: https://github.com/arelas/WeGotUsYourIP
**Live site**: https://ip.wegotussome.com

## Architecture Notes

### Theme System
- Themes are plain JSON files in `themes/`. The app fetches `themes/index.json` on load, then fetches all theme files in parallel.
- Each theme has a `base` field: `"wgus"` (card layout, JetBrains Mono) or `"terminal"` (CRT aesthetic, Courier New).
- CSS variables are applied inline via `document.body.style.setProperty()` — inline styles win over class-scoped rules.
- Light themes are auto-detected: if `page-bg` brightness > `#888`, `data-theme-light` attribute is set on body for structural tweaks.
- In terminal-base themes, `--border` controls structural panel borders; `--accent` controls text/glows. Set them equal for single-color CRT themes; use distinct values for multi-color palettes (e.g. CRT-CGA).

**When adding a new theme**, three files must be updated:
1. `themes/<id>.json` — the theme file
2. `themes/index.json` — add entry (keep alphabetical by id)
3. `sw.js` — add to `APP_SHELL` array and bump `CACHE_NAME` version

### API / Data
- Three-provider fallback chain: **ipapi.co → ipwho.is → ipinfo.io**
- Both `fetchWithFallback(ip)` and `getWhoisData(ip)` accept an optional IP. If omitted, auto-detects the user's IP.
- DNS-over-HTTPS (`dns.google/resolve`) used for PTR records and DNSBL checks — no API key needed.

### Section Toggle System
All collapsible panels (WHOIS, RDNS, Map, Blacklist, Fingerprint, CIDR Calc) are managed by the `SECTIONS` object and `toggleSection(key)`. Adding a new section means: add HTML panel, add button in `#action-buttons`, add entry to `SECTIONS`, wire up event listener.

### Permalink / Share
`?ip=x.x.x.x` in the URL auto-runs a lookup for that IP on page load and opens the WHOIS panel. The `> SHARE / PERMALINK` button copies this URL to clipboard.

### Service Worker
Cache versioning: currently **v8**. Bump `CACHE_NAME` in `sw.js` any time `index.html` or theme files change, to bust the PWA cache.

### Screenshot Script
```bash
python3 scripts/screenshot_themes.py
```
Requires `python-playwright` and chromium (`playwright install chromium`). Uses `?ip=1.1.1.1` as the demo IP. After running, commit `screenshots/*.png` and update the README gallery table if themes have changed.
